# DiscordBotLib
DiscordBotLib is an Advanced AND Basic Library for Discord Bot's, since I noticed earlier on 11/21/2025, someone was asking about for a Lib for C#, So i decided to spend about 6 hours learning more and created this project!

# Documentation

--

**DiscordBotLib.Core.Models.Activity - Activity Module**

- Handles the bots main Activity Types such as:
  - Playing
  - Streaming
  - Listening
  - Watching
  - Custom
  - Competing
- Could be used for such as:
  - Playing on {server amount} with {total server members globally}
  - Watching over {server name} join here {button with link}

Obviously I'm not going to yap on about Activity it's quite self explanitory.

**DiscordBotLib.Core.Models.Attachment - Attachment Module**

- Handles file types/uploads such as:
  - Uploaded normal files
  - Proxy URL files
  - URL files
  - Size of Files
  - and the File Name
- Could be used for such as:
  - Helpful file uploads such as downloads using a command (.download, .install, .help)
  - Upload files from the BOT instead of a user for legitimate purposes
 
**DiscordBotLib.Core.Models.Channel - Channel Module**

- Handles channels and there types such as:
  - Guild Text Channels
  - DM Text Channels
  - Guild Voice Channels
  - Group DMs (useless for bot)
  - Guild Categorys
  - Guild Announcement Channels
  - Announcement Threads & Public/Private Threads
  - Guild Stage Voice
  - Guild Directory
  - and Guild Forums
  - In this situation Guild = Server for those who don't know
- Could be used for:
  - Moderation of certain channels
  - Overview and simple to use Administration Actions
